### 使用方式

当考生想了解外省院校在广东的录取分数时：

```bash
# 查询特定院校
python3 scripts/recommend.py \
  --data-dir assets/pilot-data/guangdong-physics \
  --province 广东 --track 物理类 \
  --score 670 --rank 800 \
  --interests 计算机,电子信息 \
  --max-schools 20
```

推荐引擎会自动从 `admission_records.csv` 中匹配所有符合条件的院校（含外省），按位次排序并给出冲稳保评级。

### 外省院校数据特点

1. **数据精度**：外省院校只有省级投档层面数据(院校专业组级)，无专业级数据（除非该校官网公开）
2. **数据完整**：2023-2025三年数据完整，来源为广东省教育考试院官方投档表
3. **覆盖面广**：1462所院校覆盖全国所有在广东招生的本科院校
4. **推荐可用**：可直接用于冲稳保推荐，但专业层面需考生自行查阅目标院校招生章程

### 推荐话术

当系统推荐外省院校时，应说明：
- "该院校在广东以院校专业组方式投档，组内具体专业录取分数可能更高"
- "建议前往该校招生网查询分专业录取分数和选科要求"
- "外省院校的学费、生活成本、气候适应等也是重要考虑因素"
